# Brain-Tumor-Detection-using-CNN

The Brain Tumor Detection project aimed to develop an advanced machine learning model using Convolutional Neural Networks (CNN) to accurately classify and detect brain tumors in medical images. Leveraging the power of deep learning frameworks like Keras and TensorFlow, the project focused on enhancing the accuracy of brain tumor identification through image analysis.
